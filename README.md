# OIBSIP
Task1
This project is a beginner-friendly landing page built using HTML and CSS. It showcases fundamental web development skills such as creating layouts, structuring content with HTML, and styling elements using CSS. The landing page includes essential sections like a header with navigation links, a hero section with a call-to-action, features or services, an about section, and a footer with contact or social media links.

The purpose of this project was to practice and strengthen my understanding of core front-end concepts. I focused on using semantic HTML tags, applying CSS for styling and layout (including Flexbox and Grid), and ensuring that the page is visually appealing and easy to navigate. Special attention was given to alignment, spacing (padding and margins), color palette, font choices, and overall design aesthetics.

The project folder consists of an index.html file, a style.css file for styling, and an assets folder that contains images used on the page. To view the project, simply clone the repository, navigate to the project directory, and open the index.html file in a browser.
